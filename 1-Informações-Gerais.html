Tipos de dados

↳

Bool

Para valores booleanos (True/False)
Exemplo:

b = True
↳

Int

Para valores numericos inteiros (..., -1, 0, 1, 2, 3, ..., 12, 13, ...)
Exemplo:

i = 13
↳

Float

Para valores numericos de ponto flutuante (1.25, 2.0, -9.3, ...)
Exemplo:

f = 62.285
↳

List

Para uma sequência de dados. Os dados na lista não precisam ser de mesmo tipo array != list
Exemplo:

squares = [1, 4, 9, 16, 25]
↳

Tuple

Para uma sequência de dados. Diferente da list uma tupla é imutável, isto é, depois de criada não podemos inserir nem remover itens.
Exemplo:

t = (12345, 54321, 'hello!')
↳

Str

Para uma string. As strings são iteráveis e é possível acessar posições através do índice
Exemplo:

word = 'Python'
↳

Set e Frozenset

Para um conjunto de dados não duplicados, onde a ordem não é relevante. São iteráveis mas não permitem que seus elementos sejam acessados através de um índice. Permitem operações entre conjuntos como união e intersecção.
A diferença entre o set e o frozenset é que o set é mutável, já o frozenset é imutável. Ambos podem ser declarados porém é mais comum serem gerados a partir de uma list através da função set() ou frozenset()
Exemplo:

s = {1, 2, 3, 7, 22}
s = set([1, 22, 2, 3, 3, 7, 3, 1])

Os dois códigos geram o mesmo conjunto3

↳

Dict

Para uma sequência de dados com uma dupla de chave e valor, onde é possível acessar um valor através da chave.
Exemplo:

d = {'gatos': 5, 'cachorros': 3, 'peixes': 1}

Operadores

↳

Operadores de Comparação

Possui os operadores de sempre ↝ ==, <, >, >=, >=

O operador == além do valor testa também o tipo da variável

↳

Operadores Lógicos

Possui os operadores and, or e not

↳

Operadores de Identidade

Estes operadores são utilizados para verificar se os objetos testados referenciam o mesmo objeto

  • is ↝ referenciam o mesmo objeto
  • not is ↝ não referenciam o mesmo objeto

Exemplo:

x is y

Quando o operador == é usado em dois objetos de uma classe o operador utilizado é na verdade o is, por isso só irá retornar True se os dois objetos estiverem na mesma posição de memória, ou seja, se forem o mesmo objeto.

↳

Operadores de Associação

Estes operadores servem para verificar se determinado objeto está associado ou pertence a determinada estrutura de dados

  • in ↝ pertence
  • not in ↝ não pertence

Exemplos:

1 in [0, 1, 2, 3]
4 not in [0, 1, 2, 3]
↳

Operadores de Aritméticos

Os operadores aritméticos em python seguem o mesmo padrão que em outras linguagens, mas é importante citar algumas características relevantes:

  • / ↝ Divide dois números resultando sempre em um float
  • // ↝ Divide dois números mas o resultado respeita o tipo do dado
  • ** ↝ Exponenciação
  • Não possui os operadores '++' e '–' mas possui '+='' e '-='
↳

Operadores de Sequências

Estes são os operadores mais comuns para operações com sequências:

  • + ↝ Concatenação
  • s * n ↝ Equivalente a adicionar s a si mesmo n vezes
  • s[i:j] ↝ Fatia de s de i até j
  • s[i:j:k] ↝ Fatia de s de i até j com step k
  • min() ↝ Menor item
  • max() ↝ Maior item

Controles de fluxo

↳

If

if (x > 0):
  print("positivo")

elif (x < 0):
  print("negativo")

else :
  print("zero")

Pode haver zero ou mais partes elif, e a parte else é opcional. A palavra-chave elif é uma abreviação para else if, e é útil para evitar indentação excessiva. Uma sequência if … elif … elif … substitui os comandos switch ou case, encontrados em outras linguagens.

↳

For

O comando for do Python itera sobre os itens de qualquer sequência, na ordem que aparecem.

for w in words:
  print(w)
for i in range(5):
  print(i)
↳

While

No Python também temos o comando while para repetições, porém não há o comando do while.

while ( i > 0):
  print(i)
  i += 1
↳

Break e Continue

O comando break sai imediatamente do laço de repetição mais interno, seja for ou while.

Já o comando continue para a execução da iteração no atual momento e passa para a próxima iteração do laço.
